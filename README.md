Celem zadania było poznanie, a także implementacja Algorytmu Ewolucyjnego w rozwiązywaniu problemu **cVRP** (Capacitated Vehicle Routing Problem), a następnie porównanie jego skuteczności z algorytmami losowym oraz zachłannym, a także metaheurystyką symulowanym wyżarzaniem. 
W projekcie wykonywane było symulowane wyżarzanie. Do stworzenia algorytmów wykorzystany został język Python.
